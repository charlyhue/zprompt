# zprompt
Simple shell prompt

# Tested on
- Centos 7
- Debian 8
- Debian 9

# Install
Put 000env.sh and zprompt.sh in /etc/profile.d/.

# Configure
You can set your environement in 000env.sh and set your domain in zprompt.sh
